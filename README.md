# NotebookMT

AI-powered note-taking platform

**Live**: https://notebookmt.mthlab.com/

## Features

- User authentication (JWT)
- Notebook management
- Document upload (PDF/TXT/MD)
- AI chat (Alibaba DashScope)
- Document search
- Conversation history

## Tech Stack

- Frontend: Vue 3, Vite, TailwindCSS
- Backend: Node.js 18, Express, SQLite
- AI: Alibaba DashScope (qwen3.5-plus)
