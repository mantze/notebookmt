# 📓 NotebookMT

**AI 驅動嘅智能筆記平台** - 類似 NotebookLM

**訪問**: https://notebookmt.mthlab.com/

## ✨ 功能特點

- ✅ 用戶註冊/登入 (JWT 認證)
- ✅ 筆記本管理
- ✅ 文檔上傳 (PDF/TXT/MD)
- ✅ AI 對話 (Alibaba DashScope)
- ✅ 文檔搜索
- ✅ 對話歷史保存

## 🏗️ 技術棧

**前端**:
- Vue 3 + Vite
- TailwindCSS
- Pinia (狀態管理)
- Vue Router

**後端**:
- Node.js 18 + Express
- SQLite (better-sqlite3)
- JWT 認證

**AI**:
- Alibaba DashScope (通義千問)
- 模型：qwen3.5-plus
